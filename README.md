# 👨‍💼 Employee Management System

A full-stack **Employee Management System** built with **ReactJS** on the frontend and **Spring Boot** on the backend. This application allows you to perform full **CRUD** (Create, Read, Update, Delete) operations to manage employee records efficiently.

---

## 🚀 Tech Stack

**Frontend**
- ReactJS
- HTML5 / CSS3
- Java
- Axios (HTTP client)

**Backend**
- Java Spring Boot
- Spring Data JPA
- RESTful API
- Maven

**Database**
- MySQL (or H2 for development)

---

## ✨ Features

- 📋 View a list of all employees
- ➕ Add a new employee
- ✏️ Update existing employee details
- 🗑️ Delete an employee
- 🔍 View individual employee details
- Responsive and clean UI
